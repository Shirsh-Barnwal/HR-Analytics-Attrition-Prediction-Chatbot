# HR-Analytics-Attrition-Prediction-Chatbot
• Built an HR attrition prediction system integrating an analytics dashboard and conversational chatbot. • Trained classification model using scikit-learn on HR datasets to predict employee attrition risk. • Developed Streamlit dashboard with interactive KPI visualizations, filters, and trend insights.
